# VIC 管理画面：公開VTuber編集・おすすめ統合版

## 変更内容

- 管理タブ「公開VTuber」を「公開VTuber編集」に変更
- 独立していた「公開おすすめ」タブを削除
- 各VTuberのカード内に「おすすめ動画（件数）」を追加
- おすすめ動画一覧は開閉式で、必要なVTuberだけ読み込み
- 各おすすめ動画をその場で編集・削除可能
- VTuber情報も従来どおり編集・削除可能
- 「公開VTuber編集」の横に「更新」ボタンを追加
- 更新ボタンを押すと管理ページを再読み込み
- 公開VTuber検索で、おすすめ動画のジャンル・URL・おすすめポイントも検索対象

## GitHub更新

このフォルダ内のファイルをVICリポジトリへすべて上書きアップロードしてください。

## Apps Script更新

`AppsScript-Code.gs` の内容で現在のVIC Apps Scriptをすべて置き換え、保存後に新バージョンとして再デプロイしてください。

`setupSheets` の再実行は不要です。
