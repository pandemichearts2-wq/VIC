# VIC ジャンル別おすすめ・最大10件表示版

## 主な変更
- 「Community Recommendation」の直下におすすめ一覧を配置
- 公開中のおすすめを最大10件ランダム表示
- ジャンル選択で、そのジャンル内からランダム表示
- 「このジャンルで表示」で選択ジャンルを表示
- 初回登録・おすすめ追加の両方に動画ジャンルを追加
- 管理画面でもジャンルの確認・編集が可能

## 更新手順
1. Apps Scriptを `AppsScript-Code.gs` の内容へ置き換えて保存
2. Apps Scriptで `setupSheets` を一度実行（既存データは削除されず、ジャンル列が追加されます）
3. 「デプロイを管理」から既存ウェブアプリを新バージョンで更新
4. GitHubへ `index.html`、`app.js`、`register.html`、`register.js`、`admin.js`、`styles.css` を上書きアップロードしてコミット
5. 反映後、ブラウザで Ctrl + F5

## 既存のおすすめデータ
ジャンル列が空の既存データは、表示上「その他」として扱います。管理画面から正しいジャンルへ変更できます。

- BGMは初回から再生を試みます。ブラウザの自動再生制限がある場合は、最初の画面操作後に再生します。


## お問い合わせ機能

トップページと登録ページの右下に「お問い合わせ」を追加しています。送信内容は `VICお問い合わせ` シートへ保存され、管理画面の「お問い合わせ管理」から対応状況と管理メモを更新できます。Apps Script更新後に `setupSheets` を一度実行してください。


## 2026-07-22 更新
- 初回登録は活動名のみ必須です。その他は空白で送信できます。
- その他リンク1〜3を追加しました。
- おすすめ動画が未入力の場合はVTuber情報のみ公開され、後からおすすめを追加できます。


## おすすめ動画の重複表示

「おすすめを追加」で公開済み、または確認待ちの同じYouTube動画を送信した場合、送信せず「既に登録済みの動画です」と表示します。URL形式が違っても同じ動画IDなら重複扱いです。


## 管理画面の一括処理・件数バッジ

- 申請カードにチェックボックスを追加
- 一括チェック／一括チェック解除
- 選択した申請を一括承認／一括削除
- 申請管理タブに確認待ち件数を表示
- お問い合わせ管理タブに「未確認」と「対応中」を色分け表示
- 件数はログイン時、処理後、画面復帰時、60秒ごとに更新

一括削除は申請履歴のみを削除します。承認済みの公開VTuberや公開おすすめ動画は削除しません。
